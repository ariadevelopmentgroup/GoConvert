# Go Convert 

Go Convert-ը հզոր համակարգչային հավելված է, որը հնարավորություն է տալիս PDF ֆայլերը արագ և արդյունավետ փոխարկել տարբեր ձևաչափերի՝ DOCX, DOC, XLSX, XLS, TXT, PNG, JPG։

Ծրագիրը աջակցում է Հայերեն, Ռուսերեն և Անգլերեն լեզուներին։

Ունի պարզ և ինտուիտիվ ինտերֆեյս, որը հասանելի է յուրաքանչյուր օգտատիրոջ համար։

## 🧰 Հատկություններ

✅ **PDF → DOCX փոխարկում**
✅ **PDF → DOC փոխարկում**
✅ **PDF → XLSX փոխարկում**
✅ **PDF → XLS փոխարկում**
✅ **PDF → TXT փոխարկում**
✅ **PDF → PNG փոխարկում**
✅ **PDF → JPG փոխարկում**
🔍 OCR աջակցություն – սկանավորված էջերի տեքստի ավտոմատ ճանաչում
📊 DPI վերլուծություն – էջերի որակի գնահատում
📦 ZIP ներբեռնում – բոլոր արդյունքների միասնական փաթեթավորում
🌐 Ավտոմատ թարմացում – հավելվածը ավտոմատ թարմացվում է, առանց ձեռքով թարմացման
🎨 Ինտերֆեյս – պարզ, օգտագործելի և հարմարավետ

## 📥 Տեղադրում

1. Ներբեռնեք վերջին **`.exe`** ֆայլը [**Այստեղ**](https://github.com/ariadevelopmentgroup/GoConvert/releases)ից:
2. Տեղադրեք հավելվածը Windows օպերացիոն համակարգում։
3. Վայելեք PDF փոխարկման ամենաարագ և ամենապարզ լուծումը։

## 🧪 Կախվածություններ

Ծրագիրը անխափան աշխատեցնելու համար անհրաժեշտ է տեղադրել պահանջվող գրադարանները՝

```bash
pip install -r requirements.txt
```
