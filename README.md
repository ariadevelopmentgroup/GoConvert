# Go Convert 

Go Convert-ը հզոր համակարգչային հավելված է, որը թույլ է տալիս PDF ֆայլերը փոխարկել տարբեր ձևաչափերի՝ DOCX, DOC, XLSX, XLS, TXT, PNG, JPG։

Այն ճանաչում է՝ Հայերեն, Ռուսերեն և Անգլերեն լեզուները։

Ունի շատ պարզ և ինտուիտիվ ինտերֆեյս։

## 🧰 Հատկություններ

- ✅ **PDF → DOCX փոխարկում**
- ✅ **PDF → DOC փոխարկում**
- ✅ **PDF → XLSX փոխարկում**
- ✅ **PDF → XLS փոխարկում**
- ✅ **PDF → TXT փոխարկում**
- ✅ **PDF → PNG փոխարկում**
- ✅ **PDF → JPG փոխարկում**
- 🔍 **Ավտոմատ OCR աջակցություն** - սկանավորված էջերի տեքստի ճանաչում
- 📊 **DPI վերլուծություն** - էջերի որակի գնահատման համար
- 📦 **ZIP ներբեռնում** բոլոր արդյունքների համար
- 🌐 **Ավտոմատ թարմացման ստուգում** - Ծրագրի թարմացման կարիք չկա, այն թարմացվում է ավտոմատ
- 🎨 **Ինտերֆեյս** - օգտագործելի և պարզ ինտերֆեյս

## 📥 Տեղադրում

1. Ներբեռնեք վերջին **`.exe`** ֆայլը [**Ներբեռնել**](https://github.com/ariadevelopmentgroup/GoConvert/releases)-ից:
2. Տեղադրեք հավելվածը Windows-ում։
3. Վայելեք PDF փոխարկման ամենա պարզ և ամենա արագ ծրագիրը։

## 🧪 Կախվածություններ

Տեղադրեք անհրաժեշտ գրադարանները՝ ծրագիրը անխափան աշխատացնելու համար։

```bash
pip install -r requirements.txt
```
